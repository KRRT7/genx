Manual

Instead of producing a classical manual for GenX work is under way to make a wiki manual, where also users can contribute to improving the documentation. The manual can be found at: https://genx.sf.net/doc/. Of special intrest if you would like to get started with GenX is the IntroGuides and the FAQ. As GenX all this documentation is under construction and help is welcome.

Built in help

Note also that GenX contains a couple of help features! Go to the drop down menu Misc and choose Models Help, Plugins Help or Data loaders Help. A new window will appear that gives a short introduction, cheat sheet, for each function. This a great reference for every day work.
